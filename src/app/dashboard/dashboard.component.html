AWS Amplify with Cognito User Pool - Angular 9

This application has been created to exercise, demonstraight, and test using the AWS Amplify Javascript library with a Cognito User Pool within Angular 9. To get the most out of this you will want to perform the steps detailed below.

1 - Create an AWS Account

Creating an AWS account will require a form of payment.

  1. Open aws.amazon.com
  2. Select `Create an AWS Account`
  3. Follow the steps

2 - Create a Cognito User Pool

This example attempts to keep setup simple by working with what AWS uses sets by default when creating a User Pool. Which includes the ability for users to self register accounts.

  1. Open AWS Cognito
  2. Select `Manage User Pools`
  3. Select `Create a user pool`
  4. Enter a name in `Pool name`
    • example: My User Pool
  5. Select `Review defaults`
  6. Create the mechanisim by which the Web Application will identify itself:
    1. Select `Add app client...`
    2. Select `Add an app client`
    3. Enter a name in `App client name`
      • example: My App
    4. Uncheck `Generate client secret`
    5. Select `Create app client`
    6. Select `Return to pool details`
  7. Select `Create pool`
  8. Verification emails will only be sent if a domain is configured:
    1. Select `Domain Name`
    2. Enter a unique name in `Domain Name`
    3. Select `Save changes`

3 - Configure Amplify

The application needs to know how to connect to the User Pool. Use the `Config` option to perform this action.

4 - Exercise Communicating with the User Pool

Use the `Auth` option to interact with the User Pool leveraging the AuthClass made available within the Amplify library.
