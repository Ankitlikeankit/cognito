Cognito / Amplify / Angular 9

This application has been created to exercise, demonstraight, and test using the AWS Amplify, AWS SDK Javascript library with the AWS Cognit Service within an Angular 9 application. This application is not leveraging the Amplify CLI, this is more of an experiment in understanding what the UI developer needs to know to accessing Cognito and other AWS resources.

Local Storage, client side browser cookies, is being used to save the data needed by Amplify to access AWS Cognito. This is done to prevent the loss of the settings on browser refresh. This application is running locally in your browser, the information you input is being used by the Amplify library and as such any data transmitted is controlled by that library.

Amazon Resources:
  • Cognito
    • User Pool - Provides username/password authentication
    • Identity Pool - Provides a method to map a Group in a User Pool to an IAM Role
  • IAM
    • Role - Defines permissions to a resource
Amazon Javascript:
  • Amplify
    • Provides Objects/Methods to call a set of Amazon Resources
  • SDK
    • Provides Objects/Methods to call a larger set of Amazon Resources
Google Javascript:
  • Angular
    • Provides a frameowork to build applications
  • Material
    • Provides a set of components to build applications
